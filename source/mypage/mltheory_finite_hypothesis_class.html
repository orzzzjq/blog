As the very first example, we consider the binary classification problem, where the data points have features 𝒙i from a domain set 𝒳 and (deterministic) labels yi in {0,1}. We assume that:

  • •

    The i.i.d. assumption: The examples in the training set {𝒙1,…,𝒙m} are independently and identically distributed (i.i.d.) according to an underlying distribution 𝒟 over the domain set 𝒳.

  • •

    Finiteness of the hypothesis class: The learning algorithm choses the predictor h from a finite hypothesis set ℋ, i.e. |ℋ|<∞.

  • •

    The realizable assumption: There exists h*∈ℋ such that the true loss L(𝒟,f)⁢(h*) is zero, where f is the true labeling function (i.e. yi=f⁢(𝒙i)). This implies that if we randomly sample a data set S∼𝒟, the empirical loss LS⁢(h*) is always 0.

Say our training set is S={(𝒙1,y1),…,(𝒙m,ym)} (which is drawn from 𝒟m), and our learning algorithm is A. The output of the algorithm is the predictor h=A⁢(S), we measure the performance of h over the training set S using the empirical loss:

LS⁢(h)=|{i∈[m]:h⁢(𝒙i)≠yi}|m=1m⁢∑i=1m𝟏[h⁢(𝒙i)≠yi].

Once we obtained the predictor h, we care about its general performance in the environment (over the underlying distribution 𝒟). The generalization loss (or the true error, the risk) is measured by the probability that it predicts wrongly for a random sample:

L(𝒟,f)⁢(h)=Pr𝒙∼𝒟⁡[h⁢(𝒙)≠f⁢(𝒙)]=𝒟⁢({𝒙:h⁢(𝒙)≠f⁢(𝒙)}),

where 𝒟⁢(S) is defined as the probability that we sample a point 𝒙 in 𝒳 w.r.t. 𝒟 and the point 𝒙 lies in S.

Empirical Risk Minimization (ERM)

The objective of a learning algorithm (or a learner) is to minimize the true error w.r.t. 𝒟 and f. However, 𝒟 and f are unkown. The only thing that we can observe is the training set S∼𝒟m. Thus a natural algorithm is to minimize the empirical loss:

h=argminh∈ℋ⁡LS⁢(h).

The algorithm is known as the Empirical Risk Minimization (ERM) algorithm, which we denote as 𝖤𝖱𝖬ℋ. Though the concept is simple, ERM has a deadly weakness: it may overfit the training set. When the training set S cannot reveal the underlying structure of 𝒟, the true loss of the ERM predictor will be large. Imagine that if the labels of the samples in S are all one, the ERM predictor may predict everything as positive (though it is a very unlikely situation).

A common solution to this problem is to restrict the search space of ERM (shrink the size/space of the candidate set ℋ). To get some intuition, in the most extreme case, say if the hypothesis class ℋ is the set of all function from 𝒳 to {0,1}, ERM can learn a predictor h with zero emprical loss where h fits perfectly to the training set (h⁢(𝒙i)=yi,∀i∈[m]). However, the true loss can be arbitrarily bad, if |S|/|𝒳| is close to zero. Philosophically, if someone can explain every phenomenon, his explanations are worthless.

Though restricting the space of ℋ seems helpful in hadling the overfitting problem, it also introduced biases. Therefore, ideally we want the choice of ℋ to be based on some prior knowledge of the problem. As another extreme case, if ℋ contains only one predictor (say a threshold function), it will surely not overfit S, but the true loss is also unlikely to be small. For now, we simply assume that the realizable assumption holds, which means there exists a h*∈ℋ such that L(𝒟,f)⁢(h*)=0. Such an ℋ has perfect prior knowledge on 𝒟 and f, thus is unrealistic. We will see how to remove this assumption later, but for simplicity let’s keep it for now.

Finite Hypothesis Class

In this section we show that if the i.i.d. assumption and the realizable assumption hold, with sufficient large number of samples, the true loss of the predictor 𝖤𝖱𝖬ℋ⁢(S) learned from a finite ℋ is small:

L(𝒟,f)⁢(𝖤𝖱𝖬ℋ⁢(S))≤ε with probability at least ⁢(1-δ).

Intuitively this is true because the training set can reveal more structure of the underlying distribution with more samples. Suppose the training set S contains m samples drawn i.i.d. from 𝒟 and let hS=𝖤𝖱𝖬ℋ⁢(S), we want to upperbound the probabilty

PrS∼𝒟m⁡[L(𝒟,f)⁢(hS)>ε].

Let ℋB⊂ℋ be the set of all “bad” predictors whoes true loss is greater than ε, i.e. for each hB∈ℋB we have L(𝒟,f)⁢(hB)>ε. For each hB, we see that

Pr𝒙∼𝒟⁡[hB⁢(𝒙)=f⁢(𝒙)]=1-L(𝒟,f)⁢(hB)≤1-ε.

The probability that we learn the specific bad predictor hB is the probability that its empirical loss on S is zero:

PrS∼𝒟m⁡[LS⁢(hB)=0]≤(1-ε)m≤e-ε⁢m.

By the union bound, the probability that we learn any bad predictor hB∈ℋB is:

PrS∼𝒟m⁡[∃hB∈ℋB:LS⁢(hB)=0]≤∑hB∈ℋBPrS∼𝒟m⁡[LS⁢(hB)=0]≤|ℋB|⁢e-ε⁢m≤|ℋ|⁢e-ε⁢m,

which is equivalent to:

PrS∼𝒟m⁡[L(𝒟,f)⁢(hS)>ε]≤|ℋ|⁢e-ε⁢m.

We conclude that:

Corollary 1.

Let ℋ be a finite hypothesis class. Let δ∈(0,1) and ε>0, and let m be an integer satisfies

m≥log⁡(|ℋ|/δ)ε.

Then, for any distribution 𝒟 and for any labeling function f, if the realizable assumption holds, with probability at least (1-δ), we have

L(𝒟,f)⁢(𝖤𝖱𝖬ℋ⁢(S))≤ε.
